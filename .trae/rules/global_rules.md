# 注意事项
必须严格遵守以下规则：
- **操作系统**：Windows
- **Shell**：PowerShell, 路径分隔符使用反斜杠 `\` 或双反斜杠 `\\`
- **Python 版本**：Python 3.11，统一使用 `python` 命令执行脚本, 复制操作使用python脚本
- 严禁修改 `.git` 目录下的任何文件和配置，不得执行可能破坏 Git 历史或配置的操作
- 所有的增删改查操作（包括创建、修改、删除、重命名、移动、读取等）都只允许在当前打开的项目文件夹下进行
- 禁止修改系统文件（如 `C:\Windows`、`C:\Program Files` 等）
