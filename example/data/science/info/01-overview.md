## 网站简介

理科知识是一个基于知识立方框架搭建的示例网站，旨在系统展示数学、物理、化学、生物等理科知识。

## 内容涵盖

- **数学**：从几何到代数，探索数学的逻辑之美
- **物理**：从光学到力学，理解自然界的运行规律
- **化学**：从元素到反应，揭示物质的构成与变化
- **生物**：从细胞到生态，探索生命的奥秘

## 技术说明

本网站使用知识立方（KnowledgeCube）框架构建，数据以 Markdown 格式存储，通过编译服务生成 JSON 数据供前端展示。

本网站所有内容仅供示例展示，部分历史数据和科学描述可能不够全面，请勿作为学术研究的唯一依据。
